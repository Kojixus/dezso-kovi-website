# My Portfolio Website

My Personal portfolio site. Built with Next.js App Router and a custom UI system, optimized for fast iteration and clean presentation of experience, skills, and projects.

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- @once-ui-system/core
- CodeX
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Customize Content

Primary content lives in `app/page.tsx`. Update:

- `contact` for personal details
- `summary`, `highlights`, `competencies` for the hero and highlights
- `experience`, `leadership`, `education`, `skills`, `tools`, `certifications` for section data

## Deployment

Standard Next.js deployment works (Vercel, Netlify, or any Node host). Build then start:

```bash
npm run build
npm run start
```
